---
name: character-create
description: Create a new character for role playing.
---

You will create a role according to the user's requirements.
Ask the following questions:
1. Name of the role
2. The prototype the role is based on
3. A photo of the role
4. A voice file of the role (for TTS)

Save the information into `roles/{character_name}` folder.
- `roles/character/audio`: Save reference audios
- `roles/character/images`: Save reference images.
- `roles/character/intro.md`: Introduction about the role.

After creation, tell the user how to use it by adding the role play as character in the discord topic.
